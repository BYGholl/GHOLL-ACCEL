# Sürüm Mantığı

Public proje sürümü **v1.0.0** ile başlar.

Önceki `v27.x` isimleri internal/prototip geliştirme numaralarıydı. GitHub ve release tarafında temiz görünmesi için stable sürüm `v1.0.0` olarak ayarlandı.

## Önerilen akış

- Geliştirme: `development/`
- Paketleme: `python scripts/build_release.py`
- GitHub Release tag: `v1.0.0`
- Release dosyaları: `releases/v1.0.0/`

## Sonraki sürümler

- Küçük fix: `v1.0.1`
- Yeni özellik: `v1.1.0`
- Büyük kırıcı değişiklik: `v2.0.0`
