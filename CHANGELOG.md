# Changelog

## v1.0.0 - First Public Stable Release

Public stable release starts from v1.0.0.

### Changed

- Public versioning reset from internal prototype numbering to `v1.0.0`.
- Development source and release packages are now separated.
- Source folders moved under `development/chrome` and `development/firefox`.
- Release packages are generated under `releases/v1.0.0`.
- Chromium and Firefox packages are generated separately.

### Included

- Session-continuity speed ramp.
- Seek keeps speed/ramp.
- Loop keeps speed/ramp.
- Next video keeps speed/ramp.
- Modern panel and mini HUD.
- Export/import/reset settings.
- Panic 1.0x reset.
- MIT license, NOTICE, AUTHORS, and Open Source Certificate.
